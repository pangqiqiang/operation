Exchange 2013 模板使用方法:
Template Microsoft Exchange Server 2013共有2个Applications,Exchange 2013 Client Access和Exchange 2013 Mailbox(有交集).
针对前端服务器确保Exchange 2013 Client Access里的items是Enabled状态,只在Exchange 2013 Mailbox的items为Disabled状态.并按照实际情况进行细节调整.
